ALERT

Alert messages can be used to notify the user about something special: danger, success, information or warning.

Alert Examples

There are various type of alerts & want to add this alert in your html you need to add in each type of alert class Name alert-box and then add type of class name given below after each alert list

  • success alert   success-alert(class name)
  • warning alert  warning-alert(class name)
  • error alert        error-alert(class name)
  • primary alert   primary-alert(class name)

    • Success! Indicates a successful or positive action.

    Warning! Indicates a warning that might need attention.

    Danger! Indicates a dangerous or potentially negative action.

    Info! Indicates a neutral informative change or action.
